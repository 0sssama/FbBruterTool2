# FbBruterTool
Updated Facebook Python Brute Force Tool for Python 3

# Installation
```
pip3 install mechanize
```
```
git clone https://github.com/0sssama/FbBruterTool2/
```
```
cd FbBruterTool2
```
```
python fb-bruter.py
```
